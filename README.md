## Overview

A professional collection of animated buttons designed for modern web projects. Built with performance and flexibility in mind, Buttony UI provides developers with carefully crafted, accessible, and customizable button components.

Buttony UI offers a curated library of animated button components that enhance user interfaces with dynamic interactions. Each button is thoughtfully designed to balance visual appeal with optimal performance, making it easy to integrate sophisticated UI elements into any web project.

## Button Collections

### Buttony Original Designs

1. Glass: Translucent glassmorphism effect with backdrop blur and subtle depth
2. Minimalist: Clean, understated design focusing on clarity and elegance
3. Simple: Balanced approach with semi-transparent backgrounds and smooth transitions
